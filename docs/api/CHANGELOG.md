## 변경 기호 설명
- ✅ **추가** — 새로운 기능이 생겼어요
- 🔧 **수정** — 기존 기능의 동작 방식이 바뀌었어요
- ❌ **삭제** — 기능이 없어졌어요
- 🐛 **버그수정** — 오류를 고쳤어요
- ⚠️ **주의** — 기존에 연동된 부분에 영향이 있을 수 있어요

---

## v1.0.0 — 2026-03-08 (최초 배포)

### ✅ 추가된 기능
- **로그인** (`POST /auth/login`) — 이메일+비밀번호로 로그인, 토큰 발급
- **로그아웃** (`POST /auth/logout`) — 로그인 상태 종료
- **내 정보 조회** (`GET /users/me`) — 현재 로그인한 사용자 정보 반환
