> **마지막 수정:** 2026-03-08
> **현재 버전:** v1.0
> **담당:** 개발팀

---

## 🙋 비개발 인력을 위한 읽는 법

| 용어 | 쉬운 설명 | 예시 |
|---|---|---|
| **테이블** | 엑셀 시트 한 장. 특정 정보를 모아두는 공간 | users = 회원 목록 시트 |
| **컬럼** | 엑셀의 열. 어떤 항목을 저장하는지 | email, name, 가입일 등 |
| **PK** (Primary Key) | 각 행을 구분하는 고유번호. 절대 중복 안됨 | 회원번호, 주문번호 |
| **FK** (Foreign Key) | 다른 테이블과 연결되는 값 | 주문 테이블의 "회원번호" → 회원 테이블 연결 |
| **varchar** | 글자를 저장하는 타입 | 이름, 이메일 등 |
| **int** | 숫자를 저장하는 타입 | ID, 나이 등 |
| **datetime** | 날짜+시간을 저장하는 타입 | 가입일시, 주문일시 등 |
| **nullable** | 비워둬도 되는 항목 | 프로필 사진 (없어도 됨) |
| **NOT NULL** | 반드시 입력해야 하는 항목 | 이메일, 비밀번호 등 |

---

## 📋 테이블 목록 (한눈에 보기)

| 테이블명 | 한국어 이름 | 역할 | 연결 관계 |
|---|---|---|---|
| users | 사용자 | 서비스 가입자 정보 저장 | sessions와 연결 |
| sessions | 세션 | 로그인 상태 관리 | users와 연결 |

---

## 📌 테이블 상세 설명

### 🧑 users (사용자 테이블)
> 서비스에 가입한 모든 사용자 정보를 저장합니다

| 컬럼명 | 타입 | 한국어 설명 | 필수여부 | 비고 |
|---|---|---|---|---|
| id | int | 사용자 고유번호 | 필수 | PK, 자동증가 |
| email | varchar(255) | 이메일 주소 | 필수 | 로그인에 사용, 중복불가 |
| password | varchar(255) | 비밀번호 | 필수 | 암호화 저장 |
| name | varchar(100) | 이름 | 필수 | |
| profile_image | varchar(255) | 프로필 이미지 주소 | 선택 | nullable |
| created_at | datetime | 가입일시 | 자동입력 | |
| updated_at | datetime | 정보수정일시 | 자동입력 | |

---

### 🔐 sessions (세션 테이블)
> 사용자가 로그인한 상태를 기록합니다. 로그아웃하면 삭제됩니다

| 컬럼명 | 타입 | 한국어 설명 | 필수여부 | 비고 |
|---|---|---|---|---|
| id | int | 세션 고유번호 | 필수 | PK, 자동증가 |
| user_id | int | 사용자 번호 | 필수 | FK → users.id |
| token | varchar(500) | 로그인 인증 토큰 | 필수 | |
| expires_at | datetime | 세션 만료일시 | 필수 | |
| created_at | datetime | 로그인일시 | 자동입력 | |

---

## 🔗 테이블 관계도

```
users (사용자)
  │
  │ 1명의 사용자는 여러 개의 세션을 가질 수 있음
  │ (여러 기기에서 동시 로그인 가능)
  │
  └──── sessions (세션) [여러 개]
```
